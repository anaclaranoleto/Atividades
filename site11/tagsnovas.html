Atributos da Tag <video>

Atributo Tipo Função Principal
src URL Define o caminho do arquivo de vídeo.
controls Booleano Exibe os controles padrão (play, volume, etc.).
autoplay Booleano Inicia o vídeo automaticamente (geralmente requer 'muted').
loop Booleano Reinicia o vídeo automaticamente ao fim.
muted Booleano Inicia o vídeo sem som.
poster URL Imagem de capa exibida antes do play.
preload Texto Define o carregamento (auto, metadata, none).
width / height Número Define a largura e altura do player (px).
playsinline Booleano Impede tela cheia automática em celulares.
crossorigin Texto Configurações de CORS para domínios externos.

Atributos da Tag <audio>

Atributo Tipo Função Principal
src URL Caminho do arquivo de áudio (mp3, wav, ogg).
controls Booleano Exibe o player padrão (play, volume, barra de progresso).
autoplay Booleano Inicia o áudio automaticamente (quase sempre bloqueado se não estiver 'muted').
loop Booleano Reinicia o áudio automaticamente ao chegar no fim.
muted Booleano Inicia o áudio no mudo (silencioso).
preload Texto Dica de carregamento: auto, metadata ou none.
crossorigin Texto Permissão de CORS para áudios hospedados em outros domínios.
